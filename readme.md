# Project Workspace

This repository contains the files in the current workspace, centered around the `Vegetation-Index-VI-Monitoring-System-main` Flask application and the `recommend` support scripts.

## Main folders

- `Vegetation-Index-VI-Monitoring-System-main/`: primary vegetation index monitoring web app
- `recommend/`: soil and crop recommendation scripts plus supporting data
- `static/` and `outputs/`: local workspace folders

## Git notes

The root `.gitignore` excludes local virtual environments, caches, databases, and generated image/output files so the repository is easier to push and maintain on GitHub.
